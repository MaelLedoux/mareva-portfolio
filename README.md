# MH Interior – Portfolio

Portfolio professionnel pour **Mareva Marty-Ledoux**, architecte d’intérieur.  
Le site présente son univers, ses réalisations et son parcours, avec une approche sobre, élégante et responsive.

## ✨ Objectifs du projet

- Mettre en valeur les projets et l’identité visuelle de l’architecte
- Créer un site clair, lisible et esthétique
- Servir de support professionnel (clients, partenaires)
- Projet servant également de **terrain d’apprentissage** en développement web

## 🛠️ Technologies utilisées

- **HTML5** (structure sémantique)
- **CSS3** (Flexbox, responsive, variables CSS)
- **JavaScript (vanilla)**
  - Injection du header et du footer
  - Comportement léger de type SPA
  - Aucun framework (choix volontaire)
